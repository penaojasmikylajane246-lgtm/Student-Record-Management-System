About CICS SRMS

The CICS Student Record Management System is a role-based web platform for the College of Information and Computing Sciences at Marinduque State University.

Faculty Controls

Add, edit, delete, view, unlock student records. Full search/filter + CSV export + activity audit trail.

Student Self-Registration

Students submit their own profiles with photo. Once submitted, the record is locked — only faculty can unlock.

Secure

PDO prepared statements, bcrypt passwords, role-based access control, full activity logging.

Faculty Code

Faculty accounts require a registration code set in api.php. Default: CICS-FACULTY-2026.

Stack: PHP 8+ PDO (MySQL) · Vanilla JS Fetch API · Bootstrap 5.3 · DM Sans & DM Serif Display
